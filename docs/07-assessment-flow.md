# Assessment Flow

1. Lead intake (WordPress form, referral, inbound call, or proactive lead discovery).
2. Draft assessment created.
3. Assessor completes structured form.
4. System enforces doctrine and computes:
   - risk tier
   - staffing plan
   - asset plan
   - pricing outputs
5. Proposal generated and versioned.
6. Addendums generated for temporary expansions (if needed).
