# Client Doctrine (High-Level)

Shift model:
- Standard: **3 x 10-hour shifts per day** (supports 4-day/40-hr guard weeks and OT prevention).

Executive Protection (EP/PPO):
- PPO assigned to a specific protected person; duty of care limited to that person.
- RRT supports primaries and assumes responsibility only if formally reassigned or if primary becomes incapacitated.
