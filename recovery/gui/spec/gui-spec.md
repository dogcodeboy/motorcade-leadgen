# GUI Spec: LeadGen Recovery Kit (Mini-App)

## Phase 1 (local-first)
Buttons:
- Quick Backup
- Full Backup
- Verify Latest Backup
- Open Backup Folder
- Copy Restore Instructions

Requirements:
- user can choose backup destination every run
- remember last used destination locally
- show live logs + status
- no stored secrets

## Phase 2 (later)
Add optional remote targeting:
- select .pem
- server address + user
- test connection
- remote restore/backup

Phase 2 is not implemented in Phase 1.
